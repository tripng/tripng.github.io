Tentang Saya

Hai, Nama saya Tri Wahyu Ade Firmansyah Pananggung saya berasal dari sulawesi utara lebih tepatnya dikota Gorontalo. Saya sekarang melakukan pendidikan di Universitas Negeri Malang Jurusan Teknik Elektro program studi Teknik Informatika.

Saya memiliki hobi bermain game dan menonton anime saya lahir pada tahun 2002 bulan September tanggal 28.Saya tertarik pada dunia programming karena saat pertama kali membuat kode program sangat menarik, dan sangat menantang walaupun tantangannya sampai ketahap pusing :).

akan tetapi saya tidak menyerah dan terus mencoba hingga sekarang semester 6 saya masih tidak terlalu paham dengan apa yang saya lakukan. Dan pada akhirnya saya berhasil mendapatkan project pertama saya disaat semester 5 kemarin yaitu project membuat website undangan yang bisa dilihat di website ini
